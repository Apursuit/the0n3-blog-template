markdown 语法


## Callout

**添加了一些自定义实现的 callout 语法。**

语法：

```plaintext
> [!NOTE]
> This is a note

> [!TIP]
> Useful tip here

> [!WARNING]
> Be careful!

> [!IMPORTANT]
> Important info

> [!CAUTION]
> Dangerous action
```

效果：

> [!NOTE]
> This is a note

> [!TIP]
> Useful tip here

> [!WARNING]
> Be careful!

> [!IMPORTANT]
> Important info

> [!CAUTION]
> Dangerous action


### 原生HTML

**博客没有限制 markdown 文件里的 html 标签，因此，你可以使用 html 实现你喜欢的效果。**

html 标签

语法：

```html
<p style="color: red;">这是一个红色段落。</p>
```


效果：

<p style="color: red;">这是一个红色段落。</p>

js脚本（点击按钮显示提示）

```javascript
<button onclick="alert('Hello from Markdown!')">点击我</button>
```

效果：

<button onclick="alert('Hello from Markdown!')">点击我</button>


博客当前支持大多数常见的 Markdown 语法，以下是一些示例

### 文字

语法：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

效果：

# 一级标题
## 二级标题
### 三级标题

加粗、斜体、删除线、代码块：

语法：

```markdown
**加粗文本**
*斜体文本*
~~删除线文本~~
`代码块文本`

```

效果：

**加粗文本**
*斜体文本*
~~删除线文本~~
`代码块文本`

###  列表
 
无序列表：

语法

```markdown
- 项目 A
- 项目 B
	- 子项 B-1
	- 子项 B-2
```

效果：

- 项目 A
- 项目 B
	- 子项 B-1
	- 子项 B-2

有序列表：

语法：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：

1. 第一步
2. 第二步
3. 第三步


任务列表：

语法：

```markdown
- [x] 已完成事项
- [ ] 未完成事项
```

效果：

- [x] 已完成事项
- [ ] 未完成事项

### 链接与图片

语法：

```markdown
[访问 GitHub](https://github.com/)

![测试图片](https://the0n3.top/medias/friends/syz.png)
```

效果：

[访问 GitHub](https://github.com/)

![测试图片](https://the0n3.top/medias/friends/syz.png)

### 代码块

### 表格

```markdown

| title | 标题 | Markdown 语法测试 |
| date | 日期 | 2026-03-22 |
| tags | 标签 | 测试 |
```

效果：

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| title | 标题 | Markdown 语法测试 |
| date | 日期 | 2026-03-22 |
| tags | 标签 | 测试 |
